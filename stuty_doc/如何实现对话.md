一、对话循环：while 在哪里？
while 循环藏在 create_react_agent 内部，它的执行逻辑大致是这样的：
用户问题 "现在几点了？"
│
▼
┌─────────────────────────────────────┐
│ LangGraph 内部的 ReAct 循环 (while) │
│ │
│ ① Think: 调用 LLM，LLM 决定: │
│ "我需要调用 current_time 工具" │
│ │
│ ② Act: 执行 current_time() │
│ 返回: "2026-07-11 14:30:00" │
│ │
│ ③ Observe: 把工具结果发给 LLM │
│ │
│ ④ Think: LLM 再次思考，决定: │
│ "不需要更多工具了，直接回复用户" │
│ │
│ → 循环结束，输出最终回复 │
└─────────────────────────────────────┘
│
▼
最终回复: "现在是北京时间 2026-07-11 14:30:00"

二、记忆：怎么记住上一轮说了什么？
记忆靠两个东西配合实现：

1. MemorySaver（检查点存储器）

Python

\_checkpoint = MemorySaver() # 一个内存中的 key-value 存储
它像一个字典：{ "thread_id": [历史消息列表] }

2. thread_id（会话ID）

```python
result = await get_agent().ainvoke(
    {"messages": [HumanMessage(content=message)]},
    config={"configurable": {"thread_id": session_id}},  # ← 关键！
)
```

同一 thread_id 的调用会共享历史。LangGraph 内部流程：

````
第1轮调用 run_agent_turn("我叫张三", session_id="user_001")
    │
    ├─ LangGraph 检查: "user_001" 有没有历史？
    │  └─ 没有 → 从空白开始，加入 SYSTEM_PROMPT
    ├─ 执行 ReAct 循环
    ├─ 把整轮消息保存到 checkpointer: {"user_001": [sys, user, ai]}
    └─ 返回: "你好张三！"

第2轮调用 run_agent_turn("我叫什么？", session_id="user_001")
    │
    ├─ LangGraph 检查: "user_001" 有没有历史？
    │  └─ 有！ → 加载之前的消息 [sys, "我叫张三", "你好张三！"]
    ├─ 追加新消息: [sys, "我叫张三", "你好张三！", "我叫什么？"]
    ├─ LLM 看到完整历史，知道用户叫张三
    ├─ 执行 ReAct 循环
    ├─ 保存到 checkpointer: {"user_001": [sys, ..., "我叫什么？", "你叫张三"]}
    └─ 返回: "你叫张三！"
    ```
````
